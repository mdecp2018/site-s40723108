常見Git命令

開始工作區域

git clone：將存儲庫克隆到新目錄中

git init ：創建一個空的Git存儲庫或重新初始化現有的存儲庫

處理當前的變化

git add：將文件內容添加到索引中

git mv：移動或重命名文件，目錄或符號鏈接

git reset：將當前HEAD重置為指定狀態

git rm：從工作樹和索引中刪除文件

檢查歷史和狀態

git bisect：使用二進制搜索來查找引入錯誤的提交

git grep：打印與模式匹配的行

git log：顯示提交日誌

git show：顯示各種類型的對象

git status：顯示工作樹狀態

成長，標記和調整您的共同歷史

git branch：列出，創建或刪除分支

git checkout：切換分支或恢復工作樹文件

git commit：記錄對存儲庫的更改

git diff：顯示提交，提交和工作樹之間的更改等

git merge：將兩個或多個開發歷史記錄連接在一起

git rebase：在另一個基本提示之上重新應用提交

git tag：創建，列出，刪除或驗證使用GPG簽名的標記對象

協作

git fetch：從另一個存儲庫下載對象和引用

git pull：從另一個存儲庫或本地分支獲取並集成

git push：更新遠程引用以及相關對象
